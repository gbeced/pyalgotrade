Bitcoin Charts support
======================

The bitcoincharts package adds support for integrating with historical trade data supplied by http://www.bitcoincharts.com/
for backtesting Bitcoin strategies.

Historical trade data in CSV format is described in http://www.bitcoincharts.com/about/markets-api/, and files can be
downloaded from http://api.bitcoincharts.com/v1/csv/.
